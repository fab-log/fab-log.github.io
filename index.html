Bio

I started my professional career as a nurse. Today I'm responsible for several nursing homes as an area manager.
The lack of professional software for nursing homes made me think: "Why not write it yourself?" By that time I didn't have an idea of the efforts related to such a project. I started with excel sheets, soon added some vba, came across access and finally figured out that a web application would be the best approach. So I took a first dive into html, css and Javascript. Later on I got to know node.js and the vast possibilties of JSON as a database format. At the moment I'm in the MIT xPRO program for full stack development.
My goal is to provide clean code for user friendly applications without too many extras.
